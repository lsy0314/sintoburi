
# How to use git/github.com

## download (from github.com)
```bash
git clone https://github.com/hjoon0510/test.git
cd <folder-name>
```

## add file
```bash
touch newfile.c
git add newfile.c
```

## edit file
```bash
vi newfile.c
git diff
ls 
```

## backup (to github.com)
```bash
git commit -s (file name)
git pull
git push origin master
-your id: ****
-your password: ****
```
